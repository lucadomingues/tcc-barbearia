# 💈 Sistema Web para Agendamento em Barbearia

Projeto de Trabalho de Conclusão de Curso (TCC) desenvolvido para o curso de Tecnologia em Análise e Desenvolvimento de Sistemas.

## 📌 Descrição

Este projeto consiste em uma aplicação web para gerenciamento de agendamentos em barbearias, permitindo que clientes realizem agendamentos online, escolham serviços, efetuem pagamentos e acompanhem o status dos atendimentos.

O sistema também contará com integração com WhatsApp para notificações automatizadas e Stripe para pagamentos em ambiente de testes.

## 🚀 Funcionalidades previstas

- Cadastro e login de clientes
- Cadastro e login do administrador/barbeiro
- Agendamento de horários
- Gerenciamento de agenda
- Cadastro de tipos de corte
- Controle de duração dos serviços
- Integração com WhatsApp
- Integração com Stripe
- Cancelamento com liberação automática do horário
- Painel do cliente
- Painel administrativo
- Relatórios de atendimentos
